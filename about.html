About Custom Maps Pro

Custom Maps Pro is a powerful web application that allows you to create, manage, and share custom maps with location groups and markers.

Features

  • Create and manage location groups
  • Add markers with custom colors
  • Bulk import locations from text
  • Export data as CSV and ZIP files
  • Take map screenshots with marker lists
  • Drag and drop marker reordering

Technology Stack

  • Google Maps JavaScript API
  • Vanilla JavaScript
  • Tailwind CSS
  • Node.js & Express
  • Security best practices
